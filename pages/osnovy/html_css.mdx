# HTML + CSS

## Что такое HTML

**HTML** — это язык разметки, с помощью которого создаётся структура веб-страницы.

Он отвечает на вопрос: **что находится на странице?**

Например:

- заголовок;   
- абзац текста;   
- изображение;
- кнопка;    
- ссылка;   
- форма;   
- таблица;   
- список;   
- блок с карточкой товара;   
- меню сайта.
    

HTML не является языком программирования в строгом смысле. Он не выполняет логику вроде “если пользователь нажал кнопку, посчитать скидку”. HTML просто описывает элементы страницы.

Пример HTML:

```html
<h1>Моя первая страница</h1>
<p>Это текст на странице.</p>
<a href="https://example.com">Перейти по ссылке</a>
```

Здесь:

- `<h1>` — главный заголовок;
    
- `<p>` — абзац;
    
- `<a>` — ссылка.
    

---

## Что такое CSS

**CSS** — это язык стилей. Он отвечает за внешний вид HTML-страницы.

CSS отвечает на вопрос: **как это должно выглядеть?**

С его помощью задают:

- цвета;
    
- шрифты;
    
- размеры;
    
- отступы;
    
- расположение блоков;
    
- фон;
    
- рамки;
    
- анимации;
    
- адаптацию под телефон, планшет и компьютер.
    

Пример CSS:

```css
h1 {
  color: blue;
  font-size: 32px;
}

p {
  color: gray;
  line-height: 1.5;
}
```

Этот код говорит браузеру:

- заголовок `<h1>` сделать синим и крупным;
    
- абзац `<p>` сделать серым и удобным для чтения.
    

---

## Как HTML и CSS работают вместе

HTML создаёт **каркас страницы**, а CSS делает его **красивым и удобным**.

Можно представить так:

|Технология|За что отвечает|Аналогия|
|---|---|---|
|HTML|Структура|Стены, двери, окна дома|
|CSS|Внешний вид|Цвет стен, мебель, освещение|
|JavaScript|Поведение|Электрика, кнопки, автоматизация|

Например:

```html
<button class="main-button">Записаться</button>
```

```css
.main-button {
  background-color: black;
  color: white;
  padding: 12px 24px;
  border-radius: 8px;
}
```

HTML создаёт кнопку, а CSS делает её чёрной, с белым текстом и скруглёнными углами.

## Для чего используется HTML + CSS

HTML и CSS нужны почти в любой веб-разработке.

Они используются для создания:

- сайтов;
    
- лендингов;
    
- интернет-магазинов;
    
- личных кабинетов;
    
- блогов;
    
- админ-панелей;
    
- интерфейсов веб-приложений;
    
- email-шаблонов;
    
- страниц регистрации и оплаты.
    

Даже если сайт написан на современном фреймворке вроде React, Vue, Angular, Next.js или Nuxt, внутри всё равно используются HTML и CSS.

## Почему HTML и CSS важно знать начинающему

HTML и CSS — это база фронтенд-разработки.

Без них будет сложно понимать:

- как устроена веб-страница;
    
- почему элементы стоят не там, где нужно;
    
- как сделать адаптивную верстку;
    
- как стилизовать кнопки, формы, карточки и меню;
    
- как работать с фреймворками;
    
- как читать чужой код;
    
- как исправлять визуальные ошибки.
    

Даже если в будущем ты будешь заниматься backend-разработкой, базовое понимание HTML и CSS всё равно полезно: часто нужно понимать, как данные отображаются пользователю.

## Что такое верстка

**Верстка** — это процесс создания страницы с помощью HTML и CSS.

Например, дизайнер сделал макет сайта в Figma. Задача верстальщика или frontend-разработчика — перенести этот макет в код, чтобы страница открывалась в браузере.

Верстка включает:

- размещение блоков;
    
- подбор размеров;
    
- настройку отступов;
    
- работу с изображениями;
    
- адаптацию под разные экраны;
    
- стилизацию кнопок, форм, меню и карточек.

## Что важно изучить в HTML

Для начала стоит понять:

### 1. Основные теги

Например:

```html
h1, h2, p, a, img, button, input, form, div, section, header, footer
```

Они нужны для создания базовой структуры страницы.

### 2. Семантическая разметка

Семантика — это когда HTML-теги используются по смыслу.

Например:

```html
<header>Шапка сайта</header>
<main>Основной контент</main>
<footer>Подвал сайта</footer>
```

Это помогает браузерам, поисковикам и людям с экранными читалками лучше понимать страницу.

### 3. Формы

Формы нужны для ввода данных:

```html
<form>
  <input type="text" placeholder="Ваше имя">
  <input type="email" placeholder="Ваш email">
  <button>Отправить</button>
</form>
```

Формы используются в регистрации, авторизации, заказах, поиске, обратной связи.

### 4. Атрибуты

Атрибуты дают элементам дополнительные свойства:

```html
<img src="photo.jpg" alt="Описание изображения">
<a href="https://example.com">Ссылка</a>
```

Здесь `src`, `alt` и `href` — атрибуты.

## Что важно изучить в CSS

### 1. Селекторы

Селекторы позволяют выбрать элемент, который нужно стилизовать:

```css
p {
  color: red;
}

.title {
  font-size: 40px;
}

#main {
  background: lightgray;
}
```

### 2. Цвета, шрифты, размеры

Это основа визуального оформления:

```css
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  color: #222;
}
```

### 3. Отступы и размеры

Важно понять разницу между:

- `margin` — внешний отступ;
    
- `padding` — внутренний отступ;
    
- `width` — ширина;
    
- `height` — высота.
    

### 4. Flexbox

**Flexbox** помогает удобно располагать элементы в строку или колонку.

Например, меню:

```css
.menu {
  display: flex;
  gap: 20px;
}
```

### 5. Grid

**CSS Grid** используется для более сложных сеток.

Например, карточки товаров:

```css
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}
```

### 6. Адаптивность

Адаптивная верстка нужна, чтобы сайт хорошо выглядел на разных устройствах.

Например:

```css
@media (max-width: 768px) {
  .cards {
    grid-template-columns: 1fr;
  }
}
```

Так можно сделать, чтобы на компьютере карточки были в несколько колонок, а на телефоне — в одну.

## Как браузер использует HTML и CSS

Когда ты открываешь сайт, браузер:

1. Загружает HTML.
    
2. Понимает структуру страницы.
    
3. Загружает CSS.
    
4. Применяет стили.
    
5. Показывает готовую страницу пользователю.
    

То есть браузер превращает код в визуальный интерфейс.

---

## Чем HTML + CSS отличаются от JavaScript

|Технология|Основная роль|
|---|---|
|HTML|Создаёт структуру|
|CSS|Оформляет внешний вид|
|JavaScript|Добавляет поведение и интерактивность|

Пример:

- HTML создаёт кнопку.
    
- CSS делает кнопку красивой.
    
- JavaScript делает так, чтобы при нажатии открывалось окно, отправлялась форма или менялся текст.

## Простой пример страницы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Моя страница</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header>
    <h1>Добро пожаловать</h1>
  </header>

  <main>
    <p>Это моя первая веб-страница.</p>
    <button>Нажми меня</button>
  </main>
</body>
</html>
```

Файл `style.css`:

```css
body {
  font-family: Arial, sans-serif;
  background-color: #f5f5f5;
}

h1 {
  color: #333;
}

button {
  padding: 10px 20px;
  border-radius: 6px;
}
```

HTML описывает страницу, а CSS делает её визуально оформленной.

## Что стоит изучать после основ

После базового HTML и CSS полезно изучить:

1. **Адаптивную верстку**  
    Чтобы сайты хорошо выглядели на телефонах и компьютерах.
    
2. **Flexbox и Grid**  
    Для удобного расположения блоков.
    
3. **Работу с макетами Figma**  
    Чтобы уметь переносить дизайн в код.
    
4. **БЭМ или другую систему именования классов**  
    Чтобы CSS был аккуратным и понятным.
    
5. **Основы доступности**  
    Чтобы сайтом могли пользоваться разные люди, включая тех, кто использует экранные читалки.
    
6. **Основы SEO-разметки**  
    Чтобы страницы были понятнее поисковикам.
    
7. **CSS-переменные**  
    Удобны для цветов, размеров и единых настроек дизайна.
    

Пример:

```css
:root {
  --main-color: #000;
  --accent-color: #ff6600;
}
```

## Главное, что нужно запомнить

**HTML** — это структура страницы.  
**CSS** — это внешний вид страницы.  
Вместе они дают основу любого сайта.

Для начинающего веб-кодера HTML и CSS — один из первых и самых важных этапов. Без них сложно двигаться дальше к JavaScript, frontend-фреймворкам и полноценной веб-разработке.
